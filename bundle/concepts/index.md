# Concepts

* [OKF-Compatible LLM Wiki Bundle](okf-compatible-llm-wiki-bundle.md) - A Markdown knowledge bundle that follows OKF conventions and adds LLM Wiki extensions.
* [LLM Wiki](llm-wiki.md) - Maintained Markdown knowledge layer over raw sources.
* [RAG](rag.md) - Retrieval-augmented generation used as a page-location mechanism.
* [Company Wiki MCP](company-wiki-mcp.md) - Read-only MCP server for bundle search and reading.
* [Company Wiki Reader Skill](company-wiki-reader-skill.md) - Skill that tells agents when and how to query the bundle.
* [Image-Aware Knowledge Assets](image-aware-knowledge-assets.md) - Pattern for diagrams, screenshots, OCR, and visual evidence.
* [Progressive Disclosure](progressive-disclosure.md) - Directory-level navigation that avoids loading the whole bundle at once.

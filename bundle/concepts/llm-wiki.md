# Overview

An LLM Wiki is a curated knowledge layer between raw materials and agent answers.

It is different from dumping documents into a vector store. The Wiki captures reusable knowledge pages and relationships so agents can read stable knowledge units instead of reconstructing meaning from raw fragments every time.

# Retrieval Rule

Search results locate pages. Complete pages provide context. Source summaries provide evidence. Raw sources provide verification.

# Overview

Progressive disclosure means that every major directory can provide an `index.md` so agents and people can understand the available knowledge before reading detailed pages.

# Why It Matters

Large knowledge bases should not be loaded into context all at once. A reader should start at `index.md`, then move into `concepts/index.md`, `assets/index.md`, or other directory indexes as needed.
